# Gareth-AI
¡Bienvenido a Gareth AI! Este es un chatbot de inteligencia artificial que integra varios modelos de IA, diseñado para ofrecer una experiencia interactiva y personalizada. Actualmente, el proyecto está en fase beta y está disponible para su uso en computadoras.

Descripción
Gareth AI utiliza diferentes modelos de inteligencia artificial para conversar y responder a tus mensajes. El modelo predeterminado es GPT-4 de Microsoft, la versión más actualizada hasta 2023. Puedes seleccionar entre varios modelos disponibles en el menú del lado izquierdo.

Características
Interfaz de Usuario: Diseño moderno y responsivo para una experiencia de chat fluida.
Selección de Modelos: Cambia entre modelos de IA sin necesidad de recargar la página.
Soporte para LaTeX: Renderiza fórmulas matemáticas en LaTeX para una presentación clara de contenido técnico.
Modo Oscuro: Alterna entre un tema claro y uno oscuro según tu preferencia.
Mensajes de Bienvenida: Recibe un saludo amigable y personalizado cuando inicies la conversación.
